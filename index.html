About the App

This app is currently being developed by me currently

It brings together training schedules, communication, performance tracking, and community interaction etc into one simple, modern experience specifically for swimming
Everything listed below is a high level abstraction of the features i want to add and its only an mvp list

Built for Coaches

Coaches can create and manage their entire club directly within the app:

  • Create clubs and fully customisable squads
  • Schedule training sessions using a built-in calendar
  • Write, save, reuse, and share detailed training sets
  • Post squad-specific announcements and updates
  • View swimmer personal bests across all pool formats
  • Approve members and control access securely

Everything is organised by squad, ensuring swimmers only see whatâ€™s relevant so theres no confusion or clutter

Built for swimmers

Swimmers get a clear and focused view of training, progress etc

  • See upcoming sessions instantly on the dashboard
  • view sets for each session (if the coach writes one )
  • track pb in one place and accurately convert between long/short/yard times
  • celebrate achievements and milestones
  • stay connected with coaches and teammates
  • ability to track/register attendance maybe

Community stuff

Maybe there could also be a community space with explore pages etc for swimmers (still thinking of nuances for now):

  • Coaches can share training sets if they want to with a wider community or look at other sets and react/comment on them, and maybe export the sets so they can use themselves
  • Swimmer achievements are posted eg pbs and milestones
  • etc ...

Designed to be scalable hopefully, and i try to make sure u only see what u need to see so its not extremely confusing

all clubs are in privacy

Future Considerations

The stuff u see above is an extremely high-level overview/mvp of the app. Its smth Ive coded so far so i can visualise a foundation

Everything mentioned below will be considered/planned features

General:

  • Better and smoother UI

For Coaches

  • New club/squad level analytics section with reasoning and insights eg what worked well? How many PBs etc etc
  • Post annoucements/updates to club/squad
  • Club level/squad level customisation. Add a description? Motto? Colour theme, images, logo etc
  • History of improvements
  • A social aspect with other coaches/social media like interface
  • Write, save, reuse, download and share detailed training sets with other coaches
  • In the implementation of the social aspect, interactions with other coach's sets etc
  • Writing sets will be easier and more structured
    • Broken into toggleable sections like Warmup, Pre Main, Main, Swim Down
    • Smart/Lazy generation - if you ask AI to write a warmup to compliment an already written main set etc, or if you want AI to generate a whole set, machine learning will take into context the analytics of the squad, and write a set
    • Detection, eg if you write a 4x then the AI will shadow generate options like 100m or 200m etc
    • AI summary of a set for coaches and swimmers to understand why, taking analytics into context
    • Tags for sets, so eg Comp Warmup tag, or threshold tag
    • Customisation, if its a hard set then why not make it red?
  • Attendance tracking
  • ...

For Swimmers

  • AI will analyse and label how many calories each set is
  • Track own calories section, and monitor progress between calory gain and loss
  • Track own calories by simply typing what you ate, and AI will label amount for you
  • Analytics section
  • PB section will be able to input splits
  • PB section will have conversion buttons between long/short/yard course for all times
    • Will use smart conversion like machine learning to take into context your analytics and splits to convert
  • Able to notify coaches you cant make a training date otherwise app assumes u will attend
  • Able to show achievements like new PB or winning comps
  • Calendar integration for training
  • ...
